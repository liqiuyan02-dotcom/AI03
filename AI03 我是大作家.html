常见误区

这些想法要纠正哦！

❌ 误区1

"Trae 就是另一个聊天工具，跟之前用的差不多"

✓ 正确认知：Trae 是专业创作工具，能写故事、做网站、做游戏，聊天只是入口
❌ 误区2

"右边的预览窗口不用管"

✓ 正确认知：预览窗是"魔法镜子"，所有内容会实时排版显示
常见误区

还有这些也要注意！

❌ 误区3

"写故事直接让 AI 写就行了"

✓ 正确认知：好故事要先有设计（谁、在哪、发生什么），像盖房子先画图纸
❌ 误区4

"角色设定写得越多越好"

✓ 正确认知：简单清晰的设定比又长又乱的好，AI 也更容易理解

📝 在 Trae 里创建类型文件
帮我创建一个 Markdown 文件，标题叫"我的故事类型"。 内容是： 我选择的故事类型是：[你选的类型名称] 我选这个类型是因为：[一句话，说说为什么]
第二步 12分钟

造你的主角

每个好故事都有一个让人记住的主角

📝 角色卡模板
帮我创建一个 Markdown 文件，按照下面的模板填写我的角色设定： # 我的主角 - **名字**：[你想的名字] - **身份**：[你选的身份] - **性格**：[选两个词] - **特殊能力或特点**：[一句话描述]
🌟 示范
👦
小星
小学生 · 好奇但胆小
特殊能力：
能听懂猫说话，但别人不知道 🐱

💡 提示：每项只写一句话，越简单 AI 越容易理解

第三步 7分钟

选故事发生地

不同类型故事，发生的地方不一样

📍 地点选项
  • 🏫校园奇遇：被施魔法的学校 / 地下秘密房间
  • 🔮魔法冒险：魔法学院 / 精灵森林 / 龙之岛
  • ⏰穿越时空：古代皇宫 / 恐龙时代 / 未来城市
  • 🦁动物奇缘：会说话的森林 / 海底王国 / 大草原
  • 🚀科幻未来：火星基地 / 机器人城市 / 太空飞船
📝 在 Trae 里告诉 AI
在我的角色设定文件里，加上这两行： - **故事发生地**：[你选的地点] - **这个地方的特点**：[一句话描述]
💡 示例：龙之岛 — "一个到处都是龙的神秘小岛，岛上有一座随时会喷火的火山 🌋"
分段三 15分钟

小试牛刀

让 AI 根据你的设定，写出故事的开头

🎯 预设误概念

"AI 写的第一版就是最终版，不用改"

✓ 提醒学生：这只是"试写"，像画画前先打草稿，不满意可以改
📝 让 AI 生成开头
请根据我之前创建的设定（故事类型和角色设定），写一个故事的开头，200字以内。要求： 1. 在开头就让读者想继续看下去 2. 用 Markdown 格式排版

📝 核心任务：生成故事开头 + 练习"一句话改稿"

小游戏

一句话改稿

用你的话指挥 AI，说得越清楚，AI 做得越好

📝 可以提的修改意见
  • 💬"开头太平了，我想要更紧张刺激一点"
  • ✏️"主角的名字它写错了"
  • 🎭"我想让开头有一段对话"
  • 🖼️"描写太少了，我想要更多细节"
❌ 模糊的说法

"改好一点"、"不好看"、"再改改"

→ AI 不知道你到底要什么

✓ 具体的说法

"让主角一开始就出场"、"开头要更紧张"

→ AI 能准确理解并修改
